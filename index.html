Welcome to the NDEx Cytoscape App!

The NDEx Cytoscape App provides a mechanism for you to take any network in Cytoscape and save it to NDEx. You can also download networks to Cytoscape from NDEx. The goal of the NDEx Cytoscape App is to be both easy to install and easy to use. (And if you have any ideas for how it could be even better, we want to hear from you.)

Why Use This App?

NDEx provides a great location on the Internet where you can both save and share your networks! You can think of it as a sort of Google Drive for your networks, only with a lot more network-centric power. The NDEx Cytoscape App allows you to transfer your networks between Cytoscape, the leading desktop solution for visualizing biological networks, and NDEx.

What Sort of Networks Can I Store?

Although we have biologists in mind, any kind of network can be stored in NDEx.

Getting Started

Prerequisites:

  • 64-bit Java 7
  • Cytoscape 3.1.1 or better

Before you can install the NDEx Cytoscape App, you must have Cytoscape 3.1.1 or better installed along with Java 7. (Unfortunately, at this time Cytoscape is not guaranteed to work with Java 8, although it might.) You also must ensure that your JAVA_HOME environmental variable is set to Java 7.

  1. To install Java 7, go here: http://www.java.com/en/download/manual_java7.jsp
  2. For instruction on how to set JAVA_HOME, go here: http://www.wikihow.com/Set-Java-Home
  3. To install Cytoscape 3.1.1 or later, go here: http://www.cytoscape.org/download.html

Install the NDEx Cytoscape App

Now that you have Cytoscape installed and Java 7 setup correctly, it is time to installed the NDEx Cytoscape App. To do so, first you need to download it. The latest version is currently alpha2 and it is available for download at ftp.ndexbio.org. Just download this jar to where ever you want on your local machine.
